Pour lancer notre projet :

1.Utiliser un outil tel que MAMP
2.Ajouter le dossier de notre projet dans le dossier « htdocs » ou « www » en fonction de votre outil
3.Importer le fichier « .sql » dans phpmyadmin
4.Modifier le fichier « connexion.php » : nom de votre base de donnée, mot de passe et login
5.Lancer notre projet en ouvrant le fichier accueil.html
